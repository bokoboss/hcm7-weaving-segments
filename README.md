# เครื่องมือวิเคราะห์ส่วนทอ (Weaving Segment) ตามมาตรฐาน HCM7

เว็บแอปพลิเคชันสำหรับวิเคราะห์ส่วนทอของทางด่วน (Freeway Weaving Segments) ตามหลักการและสมการจากคู่มือ Highway Capacity Manual ฉบับที่ 7 (HCM7) บทที่ 13 พัฒนาขึ้นสำหรับวิศวกรจราจร นักวางแผน และนักศึกษา เพื่อให้การคำนวณที่ซับซ้อนกลายเป็นเรื่องง่ายและแม่นยำ

**ใช้งานโปรแกรม:** [**https://bokoboss.github.io/hcm7-weaving-segments/**](https://bokoboss.github.io/hcm7-weaving-segments/)

![ภาพหน้าจอของโปรแกรม](https://i.imgur.com/your-screenshot-url.png) ---

## ✨ คุณสมบัติเด่น

โปรแกรมนี้ได้นำหลักการจาก HCM7 บทที่ 13 มาใช้อย่างครบถ้วน และมีโหมดการวิเคราะห์ให้เลือกใช้ 3 รูปแบบ:

### 🚀 **1. โหมด Operational (วิเคราะห์ตามสภาพจริง)**
-   **วิเคราะห์สมรรถนะ:** คำนวณค่าสมรรถนะหลักของส่วนทอ เช่น ความหนาแน่น (Density), ความเร็วเฉลี่ย (Average Speed), อัตราส่วนปริมาณจราจรต่อความจุ (v/c Ratio) และระดับการให้บริการ (LOS)
-   **แสดงขั้นตอนละเอียด:** แสดงผลการคำนวณทีละขั้นตอนตามสมการในคู่มือ HCM เพื่อให้สามารถทวนสอบความถูกต้องได้ง่าย
-   **ตรวจสอบความถูกต้อง:** ตรวจสอบข้อมูลนำเข้าเบื้องต้นกับข้อจำกัดของ HCM (เช่น การเปรียบเทียบ `L_S` กับ `L_MAX`) พร้อมแสดงคำเตือนที่ชัดเจน

### 📐 **2. โหมด Design (การออกแบบ)**
-   **คำนวณเชิงออกแบบ:** แก้สมการย้อนกลับเพื่อหาค่าทางกายภาพที่เหมาะสม ให้ได้ตามระดับการให้บริการ (LOS) ที่ต้องการ
-   **หาความยาวที่เหมาะสม:** คำนวณหา "ความยาวส่วนทอขั้นต่ำ (`L_S`)" ที่ต้องการ
-   **หาจำนวนช่องจราจร:** คำนวณหา "จำนวนช่องจราจรที่ต้องการ (`N`)"
-   **ตรรกะอัจฉริยะ:** โปรแกรมสามารถจัดการกับกรณีที่ซับซ้อน เช่น เมื่อปริมาณจราจรสูงหรือต่ำเกินไป พร้อมให้คำแนะนำในเชิงวิศวกรรมที่เหมาะสม

### 📈 **3. โหมด Sensitivity (การวิเคราะห์ความไว)**
-   **แสดงผลเป็นกราฟ:** สร้างกราฟเส้นแบบอินเทอร์แอคทีฟเพื่อแสดงผลกระทบของการเปลี่ยนแปลงตัวแปรที่มีต่อสมรรถนะของส่วนทอ
-   **ทดสอบตัวแปร:** สามารถวิเคราะห์ความไวของค่าความหนาแน่นและความเร็ว ต่อการเปลี่ยนแปลงของ:
    -   ปริมาณจราจรทั้งหมด (Total Traffic Volume) โดยใช้ตัวคูณ
    -   ความยาวส่วนทอ (Segment Length)
-   **กำหนดช่วงการวิเคราะห์:** ผู้ใช้สามารถกำหนดค่าเริ่มต้น, ค่าสิ้นสุด, และจำนวนขั้นในการวิเคราะห์ได้อย่างอิสระ

## 🛠️ ข้อมูลทางเทคนิค

-   **Engine หลัก:** พัฒนาด้วย JavaScript โดยใช้สมการและตรรกะทั้งหมดจาก HCM7 บทที่ 13 รวมถึงการคำนวณตัวคูณปรับแก้สำหรับรถบรรทุก (`f_HV`), ตัวคูณปรับชั่วโมงเร่งด่วน (PHF), และความจุ (Capacity)
-   **Frontend:** สร้างด้วย HTML และตกแต่งหน้าตาด้วย **Tailwind CSS** เพื่อให้ได้ UI ที่สะอาดตา ทันสมัย และรองรับการแสดงผลบนทุกอุปกรณ์ (Responsive)
-   **กราฟ:** ใช้ **Chart.js** ในการแสดงผลกราฟที่สวยงามและตอบสนองต่อผู้ใช้ในโหมด Sensitivity
-   **ทำงานในไฟล์เดียว:** โปรแกรมทั้งหมดทำงานฝั่ง Client-side ภายในไฟล์ HTML เพียงไฟล์เดียว ไม่มีการพึ่งพา Backend ทำให้สามารถนำไปใช้งานหรือโฮสต์บนแพลตฟอร์มอย่าง GitHub Pages ได้อย่างง่ายดาย
